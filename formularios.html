Etiquetas de Formularios

Los formularios permiten recopilar información del usuario. Son fundamentales en páginas de registro, contacto y compras en línea. Un formulario bien construido mejora la experiencia del usuario y facilita el procesamiento de datos.

<form>

Definición: Contenedor principal de un formulario.

<form action="/enviar" method="post">
  <!-- campos aquí -->
</form>
  • Usar method="post" para datos privados.
  • No anidar formularios.
  • Siempre usar labels.
<input>

Definición: Campo de entrada que cambia según el atributo type.

<input type="text" name="nombre">
<input type="email" name="correo">
<input type="password" name="clave">
  • Siempre incluir atributo name.
  • Usar required para campos obligatorios.
  • Vincular con label.
<label>

Definición: Describe el campo del formulario.

<label for="nombre">Nombre</label>
<input id="nombre" type="text">
  • El for debe coincidir con el id.
  • Importante para accesibilidad.
<select>, <textarea> y <button>
<select name="ciudad">
  <option>Bogotá</option>
  <option>Medellín</option>
</select>

<textarea></textarea>

<button type="submit">Enviar</button>
  • Siempre definir el type del botón.
  • Agregar opción inicial en select.
