Etiquetas de Tablas

Las tablas permiten mostrar información organizada en filas y columnas. Son ideales para datos comparativos, horarios o reportes.

<table>

Definición: Es el contenedor principal de una tabla.

<table>
<caption>Tabla de notas</caption>

<thead>
...
</thead>

<tbody>
...
</tbody>

<tfoot>
...
</tfoot>

</table>

Buenas prácticas:

  • Agregar siempre caption para describir la tabla.
  • No usar tablas para diseño de páginas.
  • Separar la tabla en thead, tbody y tfoot.
<tr>, <th> y <td>

Definición:

  • tr → define una fila
  • th → encabezado de columna
  • td → celda de datos
<table>

<thead>
<tr>
<th>Nombre</th>
<th>Nota</th>
<th>Estado</th>
</tr>
</thead>

<tbody>

<tr>
<td>Ana</td>
<td>4.5</td>
<td>Aprobado</td>
</tr>

<tr>
<td>Luis</td>
<td>3.8</td>
<td>Aprobado</td>
</tr>

</tbody>

</table>
<caption>

Definición: Es el título de una tabla.

<table>

<caption>
Listado de estudiantes y notas
</caption>

...

</table>

Buenas prácticas:

  • Siempre incluir caption.
  • Debe ser breve y claro.
  • Ayuda a la accesibilidad.
