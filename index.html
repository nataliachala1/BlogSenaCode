Etiquetas HTML

Este blog es una guía educativa sobre las etiquetas más importantes de HTML. Está organizado en seis categorías: estructurales, texto, listas, formularios, tablas y multimedia.

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Fue creado por Tim Berners-Lee en 1991 y desde entonces ha evolucionado hasta su versión actual, HTML5.

HTML no es un lenguaje de programación; es un lenguaje de marcado que le indica al navegador cómo estructurar y mostrar el contenido de una página web. Funciona mediante etiquetas que envuelven el contenido y le dan significado semántico.

Junto con CSS (para los estilos) y JavaScript (para la interactividad), HTML forma la base de todo lo que se ve en internet.

Categorías del Blog

Este blog está organizado en 6 categorías. Haz clic en cualquiera para explorar sus etiquetas con ejemplos y buenas prácticas.

Estructurales

Etiquetas que forman el esqueleto de una página: html, head, body, header, main, footer.
Texto

Etiquetas para organizar y dar significado al contenido escrito: h1-h6, p, strong, em, span.
Listas

Etiquetas para agrupar elementos: listas desordenadas, ordenadas y de definición.
Formularios

Etiquetas para recopilar datos del usuario: form, input, label, select, textarea, button.
Tablas

Etiquetas para mostrar datos en filas y columnas: table, tr, th, td, caption.
Multimedia

Etiquetas para integrar contenido visual y auditivo: img, video, audio, iframe.
