Blog Educativo: Etiquetas HTML por Categorías

Referencia práctica con definiciones, sintaxis, ejemplos y buenas prácticas para cada etiqueta.

1. Etiquetas Estructurales

<html>

Definición: Es el elemento raíz de todo documento HTML.

Ejemplo:

<html>
  ...
</html>
  • Debe ir después de DOCTYPE.
  • Contiene head y body.

<head>

Definición: Contiene información sobre la página.

<head>
<title>Título</title>
</head>
  • Incluye metadatos.
  • No se muestra en pantalla.

<body>

Definición: Contiene todo el contenido visible de la página.

<body>
Contenido visible
</body>
  • Solo puede haber uno.
  • Aquí van textos, imágenes y tablas.

2. Etiquetas de Texto

<h1> - <h6>

Definición: Son títulos que organizan el contenido.

<h1>Título principal</h1>
<h2>Subtítulo</h2>
  • h1 es el más importante.
  • h6 es el menos importante.

<p>

Definición: Sirve para escribir párrafos de texto.

<p>Este es un párrafo</p>
  • Es la etiqueta de texto más usada.

<strong> y <em>

Definición: strong marca texto importante y em agrega énfasis.

<p>Esto es <strong>importante</strong></p>

3. Listas

<ul>

Definición: Lista con viñetas.

<ul>
<li>Elemento</li>
</ul>
  • El orden no importa.

<ol>

Definición: Lista numerada.

<ol>
<li>Paso 1</li>
</ol>
  • Se usa cuando el orden importa.

4. Formularios

<form>

Definición: Contenedor de campos para enviar datos.

<form>
...
</form>

<input>

Definición: Campo para escribir información.

<input type="text">

<label>

Definición: Texto que describe un campo del formulario.

<label>Nombre</label>

5. Tablas

<table>

Definición: Se usa para mostrar datos en filas y columnas.

<table>
...
</table>

<tr>, <th>, <td>

Definición:

  • tr = fila
  • th = encabezado
  • td = celda de datos

6. Multimedia

<img>

Definición: Sirve para mostrar imágenes.

<img src="imagen.jpg" alt="Imagen">

<video>

Definición: Permite agregar videos.

<video controls>
<source src="video.mp4">
</video>

<audio>

Definición: Permite reproducir audio.

<audio controls>
<source src="audio.mp3">
</audio>
