Etiquetas Estructurales

Las etiquetas estructurales definen el esqueleto de una página web. Sin ellas, el navegador no sabría cómo interpretar el documento. Son las primeras que debes aprender porque forman la base de cualquier archivo HTML.

<html>

Definición: Es la etiqueta raíz de todo documento HTML. Todo el contenido de la página debe estar dentro de ella.

<html lang="es">
  ...
</html>

Buenas prácticas:

  • Siempre escribir el atributo lang con el idioma.
  • No omitir el DOCTYPE antes de esta etiqueta.
<head>

Definición: Contiene información del documento que no se muestra en pantalla: título, estilos y metadatos.

<head>
  <meta charset="UTF-8">
  <title>Título</title>
</head>

Buenas prácticas:

  • Incluir siempre meta charset="UTF-8".
  • Escribir un título descriptivo.
<body>

Definición: Contiene todo el contenido visible de la página.

<body>
  <h1>Bienvenido</h1>
  <p>Contenido de la página</p>
</body>
  • Solo puede existir un body por documento.
  • No usar estilos en línea.
<header>

Definición: Representa la cabecera de la página o de una sección.

<header>
  <h1>Mi Sitio Web</h1>
  <nav>...</nav>
</header>
  • No confundir con head.
<main>

Definición: Contiene el contenido principal de la página.

<main>
  <h2>Artículo principal</h2>
  <p>Contenido importante</p>
</main>
  • Solo debe existir uno por página.
<footer>

Definición: Define el pie de página.

<footer>
  <p>© 2025 Mi Sitio</p>
</footer>
