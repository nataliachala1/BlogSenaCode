Etiquetas de Listas

Las listas permiten organizar información en grupos de elementos relacionados. Son muy usadas en menús de navegación, instrucciones paso a paso y glosarios.

<ul> - Lista desordenada

Definición: Crea una lista de elementos sin orden específico. Cada elemento se muestra con una viñeta por defecto. El orden en que aparecen los ítems no es importante para el significado.

Sintaxis básica:

<ul>
  <li>Manzana</li>
  <li>Naranja</li>
  <li>Pera</li>
</ul>

Uso adecuado: Listas de ingredientes, características de un producto, opciones de menú de navegación o cualquier conjunto de elementos donde el orden no cambia el significado.

Buenas prácticas:

  • El único hijo directo válido de ul es li.
  • Cambiar el estilo de las viñetas con CSS (list-style-type).
  • No usar listas para elementos que no sean conceptualmente una lista.
<ol> - Lista ordenada

Definición: Crea una lista de elementos numerados automáticamente. El orden de los ítems es importante y tiene significado. Por defecto usa números pero se puede cambiar con CSS.

Sintaxis básica:

<ol>
  <li>Abrir el navegador</li>
  <li>Escribir la dirección web</li>
  <li>Presionar Enter</li>
</ol>

Uso adecuado: Instrucciones paso a paso, recetas, rankings o cualquier secuencia donde el orden cambia el resultado.

Buenas prácticas:

  • Usar el atributo start para comenzar desde un número diferente al 1.
  • Usar el atributo reversed para contar en orden descendente.
  • Controlar la numeración con CSS en lugar del atributo type.
<dl>, <dt>, <dd> - Lista de definición

Definición: dl agrupa pares término-descripción. dt define el término y dd define su descripción. Es perfecta para glosarios, diccionarios o cualquier lista de pares clave-valor.

Sintaxis básica:

<dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado para crear páginas web.</dd>
  <dt>CSS</dt>
  <dd>Lenguaje para dar estilo visual a las páginas.</dd>
</dl>

Uso adecuado: Glosarios, metadatos de artículos, preguntas frecuentes y listas de términos con descripción.

Buenas prácticas:

  • Un dt puede tener varios dd asociados si el término tiene varias acepciones.
  • No usar dl solo para crear sangría visual; usar CSS para eso.
