Etiquetas de Listas

Las listas permiten organizar información en grupos de elementos relacionados. Son muy usadas en menús de navegación, instrucciones paso a paso y glosarios.

<ul> - Lista desordenada

Definición: Lista de elementos sin orden específico.

<ul>
  <li>Manzana</li>
  <li>Naranja</li>
  <li>Pera</li>
</ul>

Uso adecuado: ingredientes, características de productos o menús.

  • El único hijo válido de ul es li.
  • Cambiar el estilo de viñetas con CSS.
  • No usar listas solo para diseño visual.
<ol> - Lista ordenada

Definición: Lista numerada automáticamente.

<ol>
  <li>Abrir el navegador</li>
  <li>Escribir la dirección web</li>
  <li>Presionar Enter</li>
</ol>

Uso adecuado: instrucciones paso a paso.

  • Se puede iniciar desde otro número con start.
  • Puede usarse reversed para orden descendente.
  • Controlar estilo con CSS.
<dl>, <dt>, <dd> - Lista de definición

Definición: Lista de términos y sus descripciones.

<dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado para crear páginas web</dd>

  <dt>CSS</dt>
  <dd>Lenguaje que da estilo a las páginas</dd>
</dl>

Uso adecuado: glosarios, diccionarios o preguntas frecuentes.

  • Un término puede tener varias descripciones.
  • No usar dl solo para crear sangría.
