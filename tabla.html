5. Tablas

Las tablas permiten mostrar información en filas y columnas. Son ideales para datos comparativos, horarios y reportes. Nunca deben usarse para maquetar páginas.

<table>

Definición: Contenedor principal de una tabla. Necesita tr, th y td para mostrar datos.

Sintaxis básica:

<table>
  <caption>Tabla de notas</caption>
  <thead> ... </thead>
  <tbody> ... </tbody>
</table>

Buenas prácticas:

  • Siempre agregar caption para describir la tabla.
  • Nunca usar tables para maquetar páginas.
<tr>, <th> y <td>

Definición: tr define una fila. th define una celda de encabezado. td define una celda de datos normal.

Sintaxis básica:

<table>
  <thead>
    <tr>
      <th scope="col">Nombre</th>
      <th scope="col">Nota</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Ana García</td>
      <td>4.5</td>
    </tr>
  </tbody>
</table>

Buenas prácticas:

  • Usar scope="col" o scope="row" en th para accesibilidad.
  • No aplicar estilos directamente en td; usar clases CSS.
